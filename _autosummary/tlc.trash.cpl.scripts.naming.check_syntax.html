tlc.trash.cpl.scripts.naming.check_syntax¶

tlc.trash.cpl.scripts.naming.check_syntax(name_obj)¶

Divide el nombre del objeto en un array de 3 elementos. Comprueba si el nombre del objeto contiene mas ‘ _ ‘ de lo establecido por el pipeline, Comprueba comparando con la libreria de naming Si el primer elemento del array coincide con lo establecido por el pipeline. Comprueba si el segundo elemento coincide con las location flags establecidas por el pipeline

Parameters

name_obj (maya node) – Name of the DAG node in maya
